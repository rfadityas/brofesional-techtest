## Installation

### Clone Repo

```bash
git clone [<repository_url>](https://github.com/rfadityas/brofesional-techtest.git)
cd <repository_folder>
```

### Development

```bash
npm run install
npm run dev
```

**See on**: [http://localhost:5173](http://localhost:5173)

### Using Docker

```bash
docker compose up -d
```

**See on**: [http://localhost:3000](http://localhost:3000)
